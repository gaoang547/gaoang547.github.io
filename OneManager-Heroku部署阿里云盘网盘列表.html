OneManager+Heroku部署阿里云盘网盘列表

发表于2021-05-06|更新于2021-05-06|学习过程
|阅读量:

你是不是想和我一样把阿里云盘搭建成一个分享平台呢？像下面这样

image-20210328182902788

实现直链下载，不通过客户端，还可以分享给别人

我们先来看看实现的方法

下面有两种挂载程序（任选其一）

1.Onemanager

https://github.com/qkqpttgf/OneManager-php

2.ShareList

https://github.com/reruin/sharelist

部署方式有许多种，我们这里采用Heroku部署（要白嫖嘛~）

准备工作

  • 阿里云盘账户
  • Heroku账户
  • GitHub 项目地址

搭建步骤

我们先来讲Onemanager

首先注册一个Heroku账号（可能需要科学上网）

https://signup.heroku.com/login

image-20210328184128510

这个不用教了吧~

注册好后登陆，打开Onemanager的github项目

https://github.com/qkqpttgf/OneManager-php

点击“Deploy to Heroku”

image-20210328184305549

App name 随便填一个，下面选择美国或欧洲，然后点击“Deploy app”

image-20210328184431355

然后等它构建完成，点击“View”

image-20210328184659731

点击“点击开始安装程序”

image-20210328184754371

语言选择中文，然后点击新建API Key

image-20210328184844603

然后滑到下边找到API Key点击Reveal，复制key

image-20210328184956986

然后返回刚才的页面，粘贴key并设置管理密码

之后会进入这个页面，点击“登陆”，输入你刚才设置的密码

image-20210328185252740

然后点击“管理-设置”，然后在添加盘那里选择“Aliyundrive”，然后点击“添加盘”

image-20210328185440480

这个随便填一下

image-20210328185616863

然后点击“确认”

image-20210328185658487

接下来我们获取refresh_token

首先登陆阿里云盘

https://www.aliyundrive.com/drive

登陆阿里云盘之后，打开开发者工具，切换到Application选项卡，点开Local storage,会有一个token项，点开就可以看到refresh_token了

复制粘贴到刚才的页面，选择下面的其中之一

image-20210328190144678

这样就搭建好了，你可以在设置里美化页面，设置参数

image-20210328190232979

接下来我们讲ShareList

还是注册登陆好 Heroku

打开https://github.com/reruin/sharelist，点击“查看文档”

image-20210328190652385

点击“Deploy toHeroku”

image-20210328190832463

名字name随便起，json地址

https://raw.githubusercontent.com/reruin/sharelist/master/package.json

点击Deploy app，开始创建

image-20210328190956545

创建成功，点击“View“

image-20210328191604095

进行初始化

image-20210328191802983

要求按下面这个，虚拟路径那里选择“AliyunDrive”，然后点击保存

image-20210328191705132

下一步填入refresh_token，方法还是上面那个

image-20210328192122118
