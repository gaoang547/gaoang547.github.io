如何自制漫画

发表于2021-04-08|更新于2021-04-08|学习过程
|阅读量:

前言

虽然现在各种视频影视作品很丰富，漫画可能没有以前流行了，但相信喜欢阅读漫画的朋友依然不少。

有时出于各种原因，对现有漫画资源不满意，或者想要将自己的照片等图片制作成漫画电子书保存……

我们可能有自制漫画的需求，本文就讲解下 Kindle 漫画制作过程，将漫画图片制作成适合各种电子设备阅读的 MOBI、EPUB 等格式漫画电子书。

自制漫画教程

首先准备好漫画源图片等素材资源，大多为压缩包或图片文件夹的形式，自行搜索下载。

# ChainLP 漫画制作软件

ChainLP 是一款日本的漫画制作软件，支持将漫画图片导出为 MOBI、EPUB、PDF 等格式的电子书。

绿色软件，下载后解压到文件夹中，运行 ChainLP.exe，在菜单栏 Language 中改成英文显示，重启生效。

官网：ChainLP V0.40-17（软件需要 .Net 4.0 支持）

# 1. 整理漫画图片

确保将要使用的图片的文件名的规律性，方便导入软件后进行自动排序，如 “第1话_01.jpg”“第1话_02.jpg”。

可以使用 ReNamer 进行批量重命名。

# 2. 漫画制作

以下为最简单的制作流程，结合下图查看，只需三步即可完成（步骤 1-3 ）。漫画图片为一张一页的形式，不涉及更深入的调整。

Step 1：导入文件

首先选择导入形式：dir（目录）、arc（压缩包）、txt（文本），Role（旋转），然后点击 Input 导入。

默认第一张图片将作为封面显示。图片区 Fig 列打勾的为显示彩色图片。（Ctrl + A 全选 -> 右键 -> Color Check，可自动检测彩色图片）

Step 2：适配

设置分辨率和翻页方向。以 Kindle Paperwhite 为例，分辨率为 7581024 或 新版 10721448，选择 LockH，锁定以 “高” 为基准缩放（其他设备自行调整）。

Direc（翻页方向）选择第二个 “Left to Right”，即点击屏幕右侧翻下一页。

Step 3：导出

勾选 Fixed output fold，点击右侧三个点的按钮选择导出目录。

选择导出格式，点击 Output 导出电子书，制作完成。

# 3. 使用 KindleStrip.py 精简体积（可选）

制作成的 MOBI 格式电子书内，可能会包含一个源文件存档，导致生成的电子书体积翻倍。

我们可以使用 KindleStrip 来删除这部分多余文件，可以缩小电子书体积一倍之多，也不影响显示效果。

地址：jefftriplett/kindlestrip

首先确定电脑安装有 Python 2.7，并已加入系统环境变量。

ChainLP 高级选项使用教程

如果上面的过程无法满足自己的制作需求，或不适合自己的漫画图片情况，如一张图片包含两页漫画，可进一步调整设置。

# 页面处理

  • 2in1：两张图片合成为一张
  • Ruler：插入进度和页码显示
  • Aspect：图片分页
  • Page Analys：图片自动角度修正和裁边

如下图，一张图片包含两页漫画，我们可以调整 Aspect 将其旋转或切分。

  • 90deg / 270deg：双页旋转图片，单页正常；
  • half unit / Cover split：分页设置，将单张图片切分为两张，后面的 R2L 表示图片顺序 Right to Left，先右后左。

针对适合我们的阅读习惯的话，建议设置为：

  • Direc：Left to Right
  • Aspect：half unit R2L

但这样有点不足，就是封面可能会被从中间分开。Cover split 倒是可以保证封面在电子书中的显示效果，但需要把 Direc 改为 Right to Left，即从后往前翻页（Kindle 点屏幕左侧翻下一页）。

# 显示效果

Bold、Gamma、Sharpness 和 Histgram 都是调节显示效果的，可以改善图片显示发虚模糊的情况。

其中 Sharpness 锐度调整可以改善图片发糊的情况，让看上去的细节更清晰些。

# 批量处理

Edit -> Save Profile，保存当前配置到一个 .ini 文件，打开 BatchLP.exe，选择配置文件、输出格式和输出目录。

将文件拖进窗口中，软件会自动开始批量处理。

结语

ChainLP 这款软件虽然有点老了，但是功能很强大，而且很好用，制作出的漫画电子书效果也很不错。

ChainLP 只有 Windows 版，其他平台可以选择亚马逊官方漫画制作工具 Kindle Comic Creator，或开源软件 Kindle Comic Converter，支持 Win、Mac 和 Linux。
